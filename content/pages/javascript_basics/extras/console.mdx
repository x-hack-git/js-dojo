## コンソールの見方

`>`(左に開いた矢印)の行は、実行されるコード

```js
> a=10
```

`<`(右に開いた矢印)の行は、returnされた結果(式が評価された結果)

```js
< 10
```

何もない行は、単にconsoleに出力された情報 (`console.log`で出力された文字など)

![](/images/console_01.png)

`console.log`の戻り値は`undefined`
`var n = 10`の戻り値は`undefined`
