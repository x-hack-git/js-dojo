## テキストエディタについて

JavaScript道場では[Visual Studio Code](https://code.visualstudio.com/)の使用を推奨していますが、使い慣れたものがあればそちらを使っていただいて構いません。テキストエディタが入ってない方はインストールしておいてください。

## コメントについて
説明せずに使用していましたが、**コメント構文**というものがあります。

### コメント構文とは？
スラッシュふたつ `//`  に後続する文は**JavaScriptプログラムとして解釈されず、無視されます**。コードの説明を日本語で書きたい場合、一時的にコードを実行させたくない場合に利用します。

### コメントアウト
「ソースコードの一部をコメントにすること」です。コメントにするとコンピュータからは見えなくなります。その部分を消したのと同じ効果が得られます。

```js
// var a = 100 実行されない
var a = 110 // 実行される
```

コメントアウトの書き方は二種類あります。

```js
// 一行のコメント
　
/* もっと長い、
複数行からなるコメント
*/
　
/* ただし、 /* 入れ子のコメントは */ できず、 SyntaxError となります */
```

### コメントアウトのショートカット

多くのテキストエディタでは `コマンド(⌘) + スラッシュ(/)` の同時押しで、選択業を一括で**コメントアウト/コメント解除**することができます。
便利なので覚えておきましょう。可能な限り `/* 複数行コメント */` は使わない方が良いです。

VisualStudio以外のテキストエディタでうまく行かない場合は、テキストエディタのショートカットの設定を見てみましょう。
