## 大事なこと

関数以外の`式`と呼ばれるものも何かしら「演算した結果」を返します。
関数のreturnだけが何かしら結果を返している訳ではありません。

## returnがない関数は ”undefined” を返しています。

覚えておいてください。returnを書かない場合は ”undefined” を返します。

```js
// return のない関数
function square (num) {
    num * num
}
　
var a = square(100)
console.log(a) // undefined
```

## 式

```
> 1 + 1
< 2
> a = 10
< 10
```

## 本質的に以下の二つは同じ構造です

```js
a = 10 // a に 10 が 入る
a = func() // a に func() の実行結果が入る
```

## プログラムは右の式から順に解釈されていく

a = 1 + 1 を例に取ります

```js
a = 1 + 1 // ① 1 + 1 が実行される
a = 2     // ② 変数a に 2 が入る
2         // ③ 変数a が評価されて 2 が返る
```

## 応用するとこんなことも可能だと分かります

```js
var a, b, c
a = b = c = 10 * 10
```

```js
// a = b = c = 10 * 10 を細かくみると・・・下記と同じ意味
    　
10 * 10 // 100
c = 100 // c = 100
b = c   // b = 100
a = b   // a = 100
```
